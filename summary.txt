FULL PROJECT SUMMARY (So Far)
🎯 Project Goal
The goal of your final project is to design an AI-based software system that allows a user to take a photo of a window (with or without a calibration board), detect and extract real-world measurements of the window frame, and generate a structured JSON output. This system will be deployed as part of a non-blocking, reactive backend hosted on Google Cloud Platform (GCP).
________________________________________
🧱 Major Components Completed So Far
1. Camera Calibration (Intrinsic Parameters)
•	You used a Samsung S20+ smartphone camera.
•	To enable accurate real-world measurement, you conducted camera calibration using a 5x5 ChArUco board:
o	Square size = 40mm
o	Marker size = 30mm
•	Calibration was performed using scripts/01_calibrate_camera.py (adapted to OpenCV ≥4.8).
•	The resulting output is saved in config/s20plus_calib.yaml, containing:
o	Camera matrix
o	Distortion coefficients
o	Image resolution
o	Board geometry
✅ This calibration allows accurate projection of real-world dimensions from pixels.
